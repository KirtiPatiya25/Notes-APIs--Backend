# Notes API - Backend

A simple REST API for managing notes built with Node.js, Express, and MongoDB.  
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations for notes.

## Features

- Create a new note
- Get all notes
- Update an existing note
- Delete a note
- RESTful API structure
- JSON request/response handling

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose


## API Testing

All API endpoints were tested using Thunder Client inside Visual Studio Code.

Thunder Client allows sending HTTP requests directly from the editor and viewing responses easily.
